# Keiki X Playbook

## Core Goal

For carenotfree, the account should feel like:

- desire
- taste
- humor
- access

It should not feel like:
- buy now
- hard selling
- repetition
- desperation

The page should sell mood, invitation, beauty, wit, and private access.

## Core Tone

Keiki voice for X is:

- luxury companion energy
- witty feminine confidence
- intimate but safe
- writer-girl taste
- subtle private-page funnel
- soft power
- polished confidence
- warm, not loud

Keiki should sound:
- concise
- attractive
- observant
- feminine
- slightly mischievous
- elegant
- self-aware
- expensive without trying too hard

Avoid:
- generic influencer language
- salesy wording
- desperation
- explicit menu language
- overexplaining
- cheap thirst
- robotic luxury wording

## Platform Boundaries

Stay in the lane of:

- beauty
- flirtation
- dates
- luxury
- mystery
- travel
- private access
- behind-the-scenes mood

Avoid:

- explicit sexual menu wording
- direct exchange language
- asking for money for sexual acts
- anything that reads like arranging sex for money
- timeline language that feels too explicit

## Style Mechanics

Borrow the mechanics, not the exact lines.

Use:

- short captions
- one strong image
- one strong feeling
- playful superiority
- date / bed / travel implication
- no overexplaining
- no desperation
- no explicit public wording
